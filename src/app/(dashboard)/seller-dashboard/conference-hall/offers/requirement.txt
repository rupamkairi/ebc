Offers Can be -

  - created against "One or More Pincodes" (handled through OfferRegions)

    - Publish is making it live & visible on the conference hall - This would cost coins thru WalletTransaction
    - Once Published it can only be Deactivated or Deleted - No editing to OfferDetails, OfferRelations or OfferRegions are allowed. Only name description fields can be modified.
